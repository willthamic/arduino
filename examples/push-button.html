Pushbutton

Description

The pushbutton will allow you to power elecrtic components on and off with the push of a button. The contacts opposite of each other, vertical in this example, are always connected, but all four become connected when the button is pressed.

Materials
  • Arduino
  • Button
  • Wires
  • Led
  • Resistors
Instructions
  1. Set up the breadboard and Arduino in the same configuration as the diagram below.
  2. Start the Arduino.
  3. Press the button.
